Kick start your project development !

Getting start with your project custom requirements using a ready template which is quite difficult and time taking process, Koho Admin provides useful features to kick start your project development with no efforts !

  • Koho Admin provides you getting start pages with different layouts, use the layout as per your custom requirements and just change the branding, menu & content.

  • Every components in Koho Admin are decoupled, it means use only components you actually need! Remove unnecessary and extra code easily just by excluding the path to specific SCSS, JS file.

  • It use PUG as template engine to generate pages and whole template quickly using node js. Save your time for doing the common changes for each page (i.e menu, branding and footer) by generating template with pug.
